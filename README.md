# vue3-restaurant-management


## Project setup
```
yarn install
```

### Start Json server

```
cd db

json-server --watch database.json 
```
### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
